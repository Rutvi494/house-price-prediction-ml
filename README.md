# House Price Prediction using Multiple Linear Regression

## Overview
This project predicts house prices using Multiple Linear Regression based on various housing features.

## Problem Statement
The goal of this project is to analyze housing data and build a machine learning model that can estimate house prices accurately.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Data preprocessing
- Handling missing values
- Multiple Linear Regression
- Model training and testing
- Performance evaluation using MSE and R² Score

## Machine Learning Concepts
- Supervised Learning
- Regression Analysis
- Feature Selection
- Model Evaluation

## Results
The model was trained successfully and evaluated using regression metrics like:
- Mean Squared Error (MSE)
- R² Score

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the notebook in Jupyter Notebook or Google Colab
4. Run all cells
